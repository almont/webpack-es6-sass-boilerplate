# Webpack front-end boilerplate


### Features:
* Webpack
* ES6
* JavaScript
* SASS
* jQuery
* Font Awesome
* Bootstrap


### Installation
```
npm install
```

### Development server
```
npm run dev
```

### Production
```
npm run build
```

### Test
```
npm run eslint
```

### Fixing
```
npm run eslint-fix
```
